Site Admin

For Site Administrators: learn how to manage platform user accounts, invite individuals to be site administrators or business administrators of the platform, upload site color schemes and update css style sheets, and configure authentication domains.

Site Admin Responsibilities

Overview of the Site Administrator's capabilities and duties.

User Accounts

Manage user accounts for platform users, including activating, disabling, and deleting accounts.

Site Admins

Manage Site Administrators for the platform.

Domains

Configure Authentication Domains for APIs defined in the platform.

Business Admins

Manage Business Administrators for the platform.

Configuration

Configure basic settings for apps, APIs, connections, groups, and tickets; set up custom color schemes, define a platform login, and upload documentation files and style sheets.

Workflows

Add, change, and delete custom workflows for resources on the platform.
