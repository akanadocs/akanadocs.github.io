Contract Object

Includes information about a contract.

This object is used by the following operations:

  • POST /api/contracts: Requests a new contract between a specific version of an app and a specific version of an API.
Parameter Type Required Description
ContractID string Optional The unique ID for a specific contract between an app version and an API version.
Name Name Optional The name for the contract.
Description string Optional The text description of the contract.
APIVersionID string Optional The unique ID for a specific API version.
Endpoint string Optional The endpoint that the contract relates to.
RuntimeID string Optional Unique ID used at runtime by the app version.
Environment string Optional The environment that the contract relates to (Sandbox or Production).
Policies Policies Optional Policies relating to the contract.
State string Optional The current state of the contract. For valid values, see API Contract Status.
