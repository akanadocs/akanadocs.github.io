POST /api/contracts

Requests a new contract between a specific version of an app and a specific version of an API.

Authorization Roles/Permissions: App team member

HTTP Method

POST

Back to top

URL

https://<hostname>/api/contracts

Back to top

Sample Request

The example below shows a request for a new API contract.

Request URL

https://<hostname>/api/contracts

Sample request headers

POST /api/contracts HTTP/1.1
Host: <hostname>
Accept: application/json
Content-Type: application/json

Sample request body

{
  "APIVersionID":"apiversion23609.acmecorp",
  "RuntimeID":"gbGPCs7Uvz4VV8D0Zuvzdq6C.acmecorp"
}

Request Parameters

Response

If successful, this operation returns HTTP status code 200, with information about the contract.

Back to top

Sample Response

The sample response below returns the expanded Contract object with additional values of ContractID, Environment, and State.

Sample response headers

HTTP/1.1 200 OK
Date: Mon, 10 Jun 2013 22:15:38 GMT
Content-Type: application/json

Sample response body

{
  "ContractID": "contract24364.acmecorp",
  "APIVersionID": "apiversion23609.acmecorp",
  "RuntimeID": "gbGPCs7Uvz4VV8D0Zuvzdq6C.acmecorp",
  "Environment": "Sandbox",
  "State": "apicontract.status.activated"
}
