The Password Reset Process

The password reset process includes several steps, and several operations come into play to ensure appropriate validation of the user when resetting a password.

Below is the basic sequence of actions, including the specific operations that facilitate each step and the service that each is part of.

This Operation… Performs this Action…
POST passwordmanagement/sendPasswordResetCode Validates the email address, generates a password reset code, and sends the reset code in an email to the validated address. This operation is called when a user requests a password reset.
POST passwordmanagement/validatePasswordResetCode Validates a password reset code. This operation is called when a user is attempting to change a password after requesting a password reset code.
POST /api/login/authenticateWithPasswordResetCode Resets a user's password when the user provides a valid reset code generated with the PasswordManagement API.
